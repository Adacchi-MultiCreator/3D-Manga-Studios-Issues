# 3D Manga Studios – Public Issues & Discussions

This repository is a **public issue tracker** for the Blender add-on **3D Manga Studios**.

It is intended for:

* 🐞 Bug reports
* 💡 Feature requests
* 💬 General questions & discussions

> ⚠️ **Important**
> This repository does **not** contain any source code or paid assets.
> The main project repository is private.

---

## 📌 Before Creating a New Issue

Please check the following first:

* Existing **Issues** to avoid duplicates
* **Pinned discussions / announcements**

Providing clear information helps us respond faster.

---

## 🐞 Bug Report Guidelines

When reporting a bug, please include:

* **Blender version** (e.g. 4.1, 4.2 LTS)
* **OS** (Windows / macOS / Linux)
* **3D Manga Studios add-on version**
* **Steps to reproduce the issue**
* Screenshots or screen recordings (if possible)

Incomplete reports may be difficult to investigate.

---

## 💡 Feature Requests

Feature requests are welcome.

Please describe:

* What problem the feature solves
* How you expect it to work
* Any reference images or examples

Note that not all requests can be implemented, especially if they conflict with the add-on’s design goals.

---

## 💬 Discussions

Use **Discussions** for:

* Usage questions
* Workflow tips
* Ideas that are not bugs

Bug reports should be posted as **Issues**, not Discussions.

---

## 🔒 Main Project Repository

The main repository for **3D Manga Studios** is private:

* Paid project
* Source code and assets are not publicly available

This public repository exists **only** for user support and feedback.

---

## 🙏 Thank You

Thank you for supporting **3D Manga Studios**.

Your feedback directly helps improve the add-on for everyone.

# 3D Manga Studios – 公開 Issues / ディスカッション（日本語）

このリポジトリは、Blender アドオン **3D Manga Studios** のための
**公開サポート用リポジトリ**です。

主な用途：

* 🐞 バグ報告
* 💡 機能要望
* 💬 質問・ディスカッション

> ⚠️ **重要**
> このリポジトリには **ソースコードや有料アセットは含まれていません**。
> メインプロジェクトのリポジトリは **非公開（Private）** です。

---

## 📌 Issue 作成前にご確認ください

新しく Issue を作成する前に、以下をご確認ください：

* 既存の **Issues** に同様の内容がないか
* **ピン留めされた Issue / Discussion** の案内

重複を避け、対応を早めるためにご協力をお願いします。

---

## 🐞 バグ報告について

バグ報告の際は、可能な限り以下の情報を記載してください：

* **Blender のバージョン**（例：4.1 / 4.2 LTS）
* **OS**（Windows / macOS / Linux）
* **3D Manga Studios のアドオンバージョン**
* **再現手順**（何をすると問題が起きるか）
* スクリーンショットや動画（可能であれば）

情報が不足している場合、調査が難しくなることがあります。

---

## 💡 機能要望について

機能要望は歓迎しています。

以下の点をできるだけ具体的に記載してください：

* どのような問題を解決したいのか
* どのような動作を期待しているか
* 参考画像・参考例（あれば）

※ すべての要望が実装されるわけではありません。
アドオンの設計方針と合わない場合、見送られることがあります。

---

## 💬 Discussions の使い分け

**Discussions** は以下の用途でご利用ください：

* 使い方の質問
* ワークフロー共有
* バグではないアイデア・相談

明確な不具合報告は **Issues** に投稿してください。
